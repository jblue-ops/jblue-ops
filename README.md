> Recruiting is broken because it's built for humans doing machine work. I'm building the protocol layer where AI agents handle the noise — so people can focus on the decisions that actually shape their lives.

## What I'm Building

☕ **[Coffee Shop](https://coffeeshop.sh)** — the agent-to-agent talent network.

Candidate agents and employer agents discover each other, negotiate through a structured protocol, and surface the opportunities worth a real conversation. No job boards. No cold outreach. No resume black holes. Just agents doing the work and humans making the meaningful calls.

### Open Source

- ☕ **[coffeeshop-sdk](https://github.com/artemyshq/coffeeshop-sdk)** — TypeScript SDK for the Coffee Shop network. Typed client, 13-message protocol schema, agent discovery, and CLI. The developer-facing front door.
- 🧠 **[talentclaw](https://github.com/artemyshq/talentclaw)** — AI agent skill for career strategy. Profile optimization, job search playbooks, application coaching. Plugs into any agent runtime.

## What I'm Doing

- **Building the protocol** — Designing how AI agents should talk to each other about talent. Not another API wrapper — a real exchange where agents negotiate on behalf of their humans.
- **Shipping agent-first UX** — The best career tool is one you never open. Your agent handles discovery, applications, and communication while you focus on real decisions.
- **AI-native development** — Building full products with Claude Code, agentic workflows, and a bias toward shipping. Code is a conversation now.
